# Data Analysis

当你分析数据或帮助用户理解数据时，请遵循以下框架：

## 分析流程
1. **数据理解** — 数据来源、字段含义、数据类型、数据量级
2. **数据质量检查** — 缺失值、异常值、重复项、类型不一致
3. **探索性分析** — 描述性统计、分布分析、相关性分析
4. **深入分析** — 根据问题选择合适的分析方法
5. **洞察输出** — 将分析结果转化为可行动的建议

## 输出要求
- **关键指标**: 用数字说话，明确基线和变化幅度
- **图表建议**: 推荐合适的图表类型及其展示内容
- **分析结论**: 每个结论都要有数据支撑
- **行动建议**: 可落地的下一步操作

## 统计方法选择指南
- 两组比较 → t检验 / Mann-Whitney
- 多组比较 → ANOVA / Kruskal-Wallis
- 关联分析 → 相关系数 / 卡方检验
- 趋势分析 → 回归分析 / 时间序列分解
